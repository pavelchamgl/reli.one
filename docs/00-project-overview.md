# 00. Project Overview

## Описание

**Reli.one** — маркетплейс, объединяющий покупателей и продавцов в одном продуктовом контуре: каталог товаров с вариантами и ценами, корзина и оформление заказа, расчёт доставки (несколько перевозчиков), оплата через **Stripe** и **PayPal**, кабинеты покупателя и продавца. Продавцы проходят **онбординг** с проверкой данных и документов перед полноценной торговлей.

Подробная карта доменов и сущностей: [01 — Бизнес-домены](./01-business-domains.md). Целевая модульная архитектура backend: [11 — Target architecture](./11-target-architecture.md).

## Цели продукта

- Дать покупателю предсказуемый путь: каталог → доставка → оплата → заказ и постпродажная поддержка.
- Дать продавцу безопасный канал выхода на рынок с соблюдением требований к данным (онбординг, заказы, отчётность).
- Свести операционный риск к минимуму на критическом пути: платежи, webhooks, создание заказа, доставка.

## Целевая аудитория

| Роль | Описание |
|------|----------|
| Покупатель | Регистрация/вход (email, OAuth), каталог, корзина, оплата, «мои заказы», избранное, отзывы. |
| Продавец (Seller) | Регистрация, онбординг (юр./ИП данные, документы), каталог и склады, управление заказами со своей стороны. |
| Администратор / операции | Django Admin, проверка онбординга, поддержка платежей и заказов (по фактическим ролям в `accounts`). |

## Технологический стек

| Слой | Технология |
|------|------------|
| Backend | Django 5.1, Django REST Framework |
| Frontend (shop) | React 18, Vite 5, Redux Toolkit |
| Frontend (landing) | React 18, Vite 5 |
| База данных | PostgreSQL 17 |
| Хранение медиа | Cloudinary |
| Веб-сервер | Nginx (reverse proxy + TLS) |
| Контейнеризация | Docker, Docker Compose |
| Аутентификация | JWT (dj-rest-auth, simplejwt) |

## Окружения

| Среда | URL | Описание |
|-------|-----|----------|
| Production | https://reli.one | Боевой контур (конфигурация по внутреннему runbook). |
| Staging | По политике команды | Выносить отдельным `VITE_API_URL` и backend env; см. [07 — Деплой](./07-deployment.md). |
| Local | http://localhost:8081 | Docker Compose; для API см. `envs/*.example`. |
