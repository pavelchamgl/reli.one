## Действия по очистке git-истории

> Требует координации с командой. Все форки/клоны нужно обновить после force push.

```bash
# Установить git-filter-repo
pip install git-filter-repo

# Удалить файлы из всей истории
git filter-repo --path backend/backend/www.solopharma.shop.key --invert-paths
git filter-repo --path backend/backend/www.solopharma.shop.crt --invert-paths
git filter-repo --path envs/database.env --invert-paths
git filter-repo --path envs/backend.env --invert-paths

# После — force push (требует разрешения на ветке)
# git push origin --force --all
```
