# Release Checklist Skill

Use this skill before merging or deploying.

Checklist:
1. Tests pass.
2. Frontend builds.
3. No secrets added.
4. No unintended migrations.
5. API contracts preserved or documented.
6. Critical flows checked:
   - auth
   - catalog
   - checkout
   - payment
   - delivery
   - order
   - seller onboarding
7. Deployment notes updated.
8. Rollback risk identified.
