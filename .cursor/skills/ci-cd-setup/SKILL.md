# CI/CD Setup Skill

Use this skill when creating or improving CI/CD.

Rules:
1. Start with CI only.
2. Do not deploy to production automatically.
3. Run backend tests.
4. Run migration check.
5. Run frontend lint/build where scripts exist.
6. Keep workflow simple.
7. Use GitHub Actions.
8. Document commands in docs/07-deployment.md and docs/08-testing-strategy.md.
