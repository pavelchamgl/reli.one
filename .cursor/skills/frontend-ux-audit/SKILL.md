# Frontend UX Audit Skill

Use this skill when analyzing or improving frontend usability.

Process:
1. Do not rewrite UI immediately.
2. Identify user flow.
3. Inspect pages, components, API calls, forms, validation, loading/error states.
4. Check:
   - registration/login usability
   - catalog browsing
   - product detail clarity
   - checkout steps
   - delivery selection
   - payment handoff
   - order confirmation
   - seller onboarding
5. Produce recommendations with priority:
   - blocks conversion
   - confuses user
   - visual polish
6. Keep API contracts unchanged unless explicitly requested.
